===============================================================
AI驱动的大学生职业规划平台 — 完整技术栈清单
===============================================================

【一、前端技术栈】

跨平台框架    | uni-app 3.0.0-408          | 一套代码编译到微信小程序/H5/App等多端
核心框架      | Vue.js 3.4.21             | 响应式 UI 框架 (Composition API)
开发语言      | TypeScript 4.9.4          | 类型安全的前端开发语言
构建工具      | Vite 5.2.8                | 极速热更新的前端构建器
国际化        | vue-i18n 9.1.9            | 多语言支持
类型检查      | vue-tsc 1.0.24            | Vue 单文件组件的 TypeScript 校验
目标平台      | 微信小程序 (mp-weixin)     | 主要发布目标

【二、后端技术栈】

核心框架      | Spring Boot 3.5.3         | Java 后端一站式开发框架
编程语言      | Java 17 (LTS)             | 长期支持版本
Web 层       | Spring MVC                 | RESTful API 路由与控制器
参数校验      | Spring Validation (JSR-303)| @NotBlank、@Size 等输入安全校验
切面编程      | Spring AOP                 | 统一请求日志切面 (WebLogAspect)
ORM 框架     | Spring Data JPA + Hibernate | MySQL 实体映射与 CRUD
文档数据库    | Spring Data MongoDB         | 非结构化文档存储（测评、简历、聊天记录）
缓存层        | Spring Data Redis          | 用户进度缓存、会话状态
鉴权安全      | JJWT (io.jsonwebtoken) 0.11.5 | 无状态 JWT Token 签发/验签/RBAC
API 文档     | Knife4j (OpenAPI 3) 4.5.0   | Swagger 在线接口文档
代码简化      | Lombok                     | @Data、@Builder 等注解减少样板代码
JSON 处理    | Jackson + JSR310            | 序列化/反序列化，Java 8 时间类型支持
单元测试      | Spring Boot Test + H2      | 内存数据库驱动的集成测试
构建工具      | Maven                      | 依赖管理与项目构建

【三、数据库与存储】

MySQL         | 关系型主数据源（用户表、面试记录、职业路径节点表）
MongoDB       | 文档型存储（测评结果、简历详情、聊天会话历史）
Redis         | 缓存层（用户进度缓存、热点数据加速）
阿里云 OSS    | 对象存储（简历 PDF 文件上传与管理）

【四、AI / 大模型集成】

阿里云通义千问 (Qwen-Max) | 基座大模型，经由 DashScope API 调用
SSE (Server-Sent Events) | 实现流式逐 Token 输出（打字机效果）
Prompt Engineering       | 面试官人设系统提示词、简历诊断模板、职业助手角色设定

【五、DevOps / 工程化】

Git           | 版本控制（15 个 feature 分支的敏捷模拟流）
Gitee         | 远程代码托管 + Pull Request 管理
飞书 (Feishu) | 敏捷看板 + Sprint 任务追踪
微信开发者工具 | 小程序模拟器编译与真机调试
